=================================
Welcome to dripy's documentation!
=================================
dripy is a python package for analyzing drilling data.  It is currently heavily focused on data from Pason. 

Getting Started
================

Requirements
------------
* Python 3.6

Installation
-------------

You can do a direct install via pip by using:

.. code-block:: cmd

    > pip install dripy
